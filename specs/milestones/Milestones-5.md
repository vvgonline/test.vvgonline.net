# Milestone 5: Data Visualization with Charts

## Objectives
- Integrate charts for facts and figures.
- Display key statistics on relevant pages.

## Tasks

### 5.1 Data Preparation
- [ ] Create or update `facts-figures.json` under `wwwroot/data/`.
- [ ] Define schema for the data (labels, values, series, etc.).

### 5.2 Chart Component
- [ ] Implement `Chart.razor` with JS interop to a chart library (e.g., Chart.js).
- [ ] Create a C# model for chart data to keep bindings clean.
- [ ] Support multiple chart types as needed (e.g., bar, line).

### 5.3 Chart Integration
- [ ] Add charts to the home or dedicated statistics page.
- [ ] Ensure charts respect theme (colors and background).
- [ ] Test charts on mobile and desktop for responsiveness.
